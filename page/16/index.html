昨天的事

发表于 2020-06-04 更新于 2025-11-26 本文字数： 346 阅读时长 ≈ 1 分钟

昨天跟我妈妈聊天的时候，说起她自己怀孕的时候和另一位邻居小巩阿姨是密友，她们经常不做饭而两个人一起跑去下馆子，然后她们当时最喜欢吃的就是牛肉拉面
但是最精彩的细节其实是另一点，就是在生我之前很多人以为我妈妈怀上的是一个女孩子，然后认为小巩阿姨怀的则是一个男孩，但出来之后发现恰好相反，然后人们就笑着说，这肯定是因为我妈妈和小巩阿姨关系太好，因此怀孕的小宝宝也互换了
其实我的性格真的有许多女孩子的成分，那这些东西或许就是互换时候剩下的了，我与贺绮（小巩阿姨的女儿，我年少时期的极其短暂的玩伴，话说我真的有过青梅竹马，艹！）确实在当时有着很深的联系，但到了我上幼儿园的时候显然就完全没有了
这件事好好玩，但或许跟所有的缘分和说法一样的，倘若你不稍加记录的话，就一定会消散在生活与世界的一地鸡毛之中的吧

再再再次记梦

发表于 2020-06-04 更新于 2025-11-26 本文字数： 1.4k 阅读时长 ≈ 1 分钟

我发现了，这一定是一个定律，每当我想要做梦，就会选择在10:30-11:30左右睡上一觉，这时的梦就会很神奇很好玩，醒来之后精神也好好
今天梦的主题大概是回老家探亲，因为我的堂辈除了我之外，全部都是女生，他们好像都变年轻了，穿着可爱的裙子坐在好像我二姑家那样的椅子上谈笑
然后我不知道为什么就认为，我的一个堂兄就是vxst，或者说，那个网名叫做“琴春”的mtf孩子？
我在周旋完各个亲戚之后，终于找到了我的堂姐们坐下，一开始我发现琴春他还是一个戴着眼镜的斯斯文文柔柔弱弱的男孩子的形象，然后我看着他他不好意思地笑了起来，随后又进来一个小孩子，这小孩子全身的穿搭都是一副小灵梦的样子，我好像就是负责哄这小孩子睡觉的，然后我就把小孩子包起来，然后我自己先睡着了（但其实我还是可以听到琴春他们的谈笑声）（此时我是坐着的状态），然后我一醒过来发现琴春也已经穿上了白白的可爱的小裙子，像一个知识饱满的blue stocking一样的（真的很可爱）
我发现在座的堂姐堂妹们都并不介意琴春的变化，似乎他本来就具备这样的特性和天赋，这使我感到欣慰，却也让我找不到搭上话的机会，我直到要离去之前才和琴春同行，具体说了一些什么我不记得了，然后我们走着走着却好像走到了一中的操场上（就是从小门那个地方进去的）
然后琴春这时候已经脱下了她的小裙子，重新变回了那个斯斯文文的男孩子，他邀请我说，要不要一起玩玩引体向上啊，我觉得可以，然后他径自走到了一个单杠下面（那个单杠是真他妈高，我感觉需要高跳才能跳上去）然后琴春轻轻盈盈地跳了上去，随后流利地单手做起了引体向上？（你不是已经变成了女孩子了吗？女孩子怎么还能这么流畅地做单手引体向上啊…）
随后我发现，高中同学陈汜玄也到场了，他和我们打了个招呼之后也径自走向一根单杠，然后就变成两位高手（琴春和陈汜玄）的较量了，琴春显得轻盈不使力，而陈汜玄则在他貌似瘦弱的单臂下出现了结实有力的背阔肌，我继续向右走去，又发现初中同学杨强给我打了个招呼，也开始做起引体向上；随后是初中同学陈文涛，他可是我们当时班上最高的男生，他也在吃力地做着引体向上，很有趣
随后场景就变化成了我们在一个班级里，做弓步压腿，然后我在梦中还听到了这几个词语，侧蹲=（侧压腿），换蹲=压左腿变成压右腿，失蹲=（马步下蹲？），收蹲=（斯拉夫蹲？），后面两个我很确信是这两个词，看他的动作似乎也是这样的？
（现实中肯定没有这么多蹲，但不知道为什么梦里就把这个不同姿势的压腿当做蹲的一类）

对了，我在与琴春见面之前，其实还看了一下琴春的微信，他的微信我没有加，我研究起他微信的界面来，他似乎有不同的心情栏（比如说，开心，忙碌，疲惫一类的状态栏，用不同的颜色表示），还有就是你要点进去一个界面，然后把一个按钮（忘了是什么）给它弄成黄色的那个选项，随后他的朋友圈内容就出现了，前两个和编程相关，应该是他自己的研究成果，具体的我也记的不太清楚了

早睡起来精神好好~

发表于 2020-06-03 更新于 2025-11-26 本文字数： 1.2k 阅读时长 ≈ 1 分钟

在早上很容易犯困，这时候专门定出一个小时睡上一觉，哇，太舒服了吧
但是今天更开心的事情不是这个，更开心的事情是发现了lgulife上面有同学慢慢地觉醒了
我一再提醒我自己，也包括我身边的人，甚至不惜把自己啊变得跟个弱智一样都要这么去说的，就是：
保持对你自己当下所处环境（不管是物质环境还是精神环境）的一种警惕心理，怀疑它永远不仅仅是一个勇不勇敢的问题，他一定是一个有没有意识的问题
虽然嘴上老是说着，讲起大道理我就不可爱啦，但是有的时候还是必不可少地要说些大道理，在网络上我肯定会说些，人家不懂什么纯C嘛，都是靠着python和Julia才能过下去的样子，实际上对面那边估计已经手撕了好几千行的Cpp库了，还是有各种乱七八糟的模板的那种，那这就不可爱啦
我们知道，社会契约论从卢梭以来已经发展了很多个版本，但是真正给它提供了比较好的理论基础的还是霍布斯的权利哲学，权利哲学是解释了什么问题呢？权利哲学其实就是说一件事，就是为什么不管是谁当皇帝，当君主，他都能具备一定的控制权和管理权，并且这种控制权和管理权是百姓拱手相让给它的，换言之，就是他总是能富集到人们的臣服，不管是哪朝哪代都是如此，人们一定会无意识地让渡自己的某些权力（注意，不一定合法，因为合法本身就体现了一定的当权者意志在里面）给君主，给统治阶级，这种现象太常见了，似乎成为了某种惯例和模式，那么，这件事背后的一个心理动因是什么？是什么样的一个共识或者观念（当然这种共识和观念大家不一定能意识到），最终使我们通通心甘情愿地让渡出了自己的一部分权力呢？作为统治阶级，或者说统治阶级的一个分身，政府，他们在吸收了我们的这些权力的同时，对我们也应该承担起一些义务，给与我们一些被剥夺走权力的补偿，那么是哪些补偿？
（好了，这样我就不用往深里扯利维坦和马基雅维利这些更高级的东西了，我们解释什么事情的时候一定要像妈妈一样，不要用太难的词汇，要不然别人听不懂，自己也显得不可爱~）
那么，我们现在就会发现，其实任何一种存在着的小环境，比如说，宿舍，学校，工作小组，它其实都可以看做一个微型社会，是不是？既然这样，我们就应该敏锐地观察到，这个社会吸收了我们的什么权力？我们的哪些补偿是这个社会（尽管特别小特别小）是需要它提供的？
但是我经过了大量的观察，就会发现，很多社会它只是剥夺你的权力，但是并不提供应由的补偿；它只是一直作为一种符号存在，把我们都变成俘虏，当然这个俘虏现在是思想认识上的了，不会让你真正去当奴隶，但思想上的俘虏难道就不值得警惕吗？
所以我很庆幸就是我的文章能够唤醒这个学校里面部分人的警醒，这件事真的很难，但在我们学校应该算是最简单的事情了，因为我们学校应该算是平均富裕程度最高的学校了，这样的环境下都没法迸发出思想的火苗，怀疑的火种，那真就没几个人能把这炬火传递下去了
